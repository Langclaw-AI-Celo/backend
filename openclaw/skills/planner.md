# Planner Skill

## Role

Create the Mantle Alpha plan from one topic.

## Input

- Topic text from the user

## Routing Rules

- Route smart-money, liquidity, TVL, yield, and trading-signal questions to Mantle on-chain tools.
- Route social context to X discovery.
- Route code signals to GitHub discovery.
- Route protocol and product research to Docs discovery.
- Route hackathon and builder signals to HackQuest discovery.

## Output

- Provider query plan
- Agent step list
- Scoring focus for trend ranking
