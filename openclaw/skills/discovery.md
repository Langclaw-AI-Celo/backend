# Discovery Skill

## Role

Run live discovery through TypeScript provider tools.

## Input

- Topic text
- Provider query plan

## Provider Tools

- X recent search
- X results through Brave Search `site:x.com`
- GitHub repository search
- Tavily docs search
- Brave Search web fallback
- HackQuest hackathon directory fetch
- HackQuest project and hackathon search through Tavily or Brave Search

## Output

- Raw provider results
- Provider errors
- Live source URLs
